Приложение со списком пользователей и детальной информацией.

## Функциональность
- Загрузка списка пользователей с API
- Просмотр детальной информации (фото, должность, компания, город)
- Поиск по имени
- Тёмная/светлая тема
- Адаптивный дизайн
- Клавиатурная навигация
- Анимации и скелетон-лоадер

## Технологии
- React (функциональные компоненты)
- TypeScript
- Vite
- CSS Modules
- GitHub Pages

## Установка и запуск
```bash
npm install
npm run dev
